# DreamX Studio — 页面交互设计

## 页面地图

```
/ (首页，未登录)
/projects (项目列表)
/canvas?projectId=xxx (画布编辑器 — 核心页面)
/canvas?projectId=xxx&episodeId=xxx (单集详情)
/characters?projectId=xxx (角色管理)
/assets (素材库)
/redbook (小红书图文转视频)
/tasks (任务中心)
/settings (设置)
/subscription (订阅)
```

## 3. 画布编辑器 `/canvas`（核心页面）

### 布局：三栏

```
┌────────┬──────────────────────┬──────────┐
│ AI 聊天 │     React Flow 画布   │ 属性面板  │
│ (可折叠) │                      │ (可折叠)  │
│        │  [CheckPoint]         │          │
│ ┌────┐ │       ↓               │ 节点标题  │
│ │对话│ │  [StoryBible]         │ ──────── │
│ │历史│ │       ↓               │          │
│ │    │ │  [CharacterPack]      │ 属性表单  │
│ │    │ │       ↓               │ 内容区域  │
│ │    │ │  [PlanningCenter]     │          │
│ │    │ │       ↓               │          │
│ │    │ │  [Script]             │ [操作按钮]│
│ │    │ │       ↓               │          │
│ │    │ │  [SceneDesign]        │          │
│ │    │ │       ↓               │          │
│ │    │ │  [SegmentDesign]      │          │
│ │    │ │       ↓               │          │
│ │    │ │  [Compose]            │          │
│ └────┘ │                      │          │
│        │ [MiniMap] [Controls]  │          │
│ [输入框]│ [画布视图|时间线视图]  │          │
└────────┴──────────────────────┴──────────┘
```

### 节点详情（点击节点 → 右侧面板展开）

#### CheckPoint 节点
- 语言选择（中文/英文）
- 画面比例（9:16 / 16:9 / 1:1）
- 内容评级（G / PG / PG-13 / R）
- 视觉风格选择器（网格预览图，100+ 种）
- 单集时长设置

#### StoryBible 节点
- 故事标题
- 世界观描述（可编辑文本区）
- 核心冲突
- 主题
- AI 生成按钮 / 手动编辑切换
- 生成状态指示器（Lottie 动画）

#### CharacterPack 节点
- 角色卡片网格：
  - 头像（AI 生成/上传）
  - 名字 + 角色类型标签（主角/配角）
  - 性别 / 年龄
  - 配音选择下拉
- [+ 添加角色] 按钮
- [AI 自动提取] 按钮
- 点击角色卡片 → 展开 CharacterBible 编辑

#### PlanningCenter 节点（连续剧集模式）
- 概览 Tab：项目统计
- 封面 Tab：AI 生成封面 + 编辑/重做
- 剧集列表 Tab：每集缩略图 + 摘要
- 点击「查看详情」→ 进入单集画布

#### Script 节点
- 分场剧本 Markdown 渲染
- 场景列表：场景标题 + 地点 + 时间
- 每场景可展开：描述 + 对白 + VO
- [AI 生成剧本] / [导入剧本] 按钮
- 内联编辑（点击即编辑）

#### SceneDesign 节点
- 场景卡片列表
- 每场景：视觉描述 + 情绪标签 + 情绪强度滑块
- BGM 建议 + 转场效果选择
- 时长调整

#### SegmentDesign 节点
- 分镜宫格预览（2×2 / 3×3）
- 每镜头卡片：
  - 缩略图（生成后显示）
  - 景别/角度/运镜 三联选择器
  - 镜头描述（可编辑）
  - Motion Prompt 编辑器（三层）
  - [生成图像] [生成视频] 按钮
  - 状态指示（pending/generating/done）
- 镜头语法可视化选择器：
  - 景别：ECU → CU → MCU → MS → MLS → LS → ELS（图标+示意图）
  - 角度：平视/仰拍/俯拍/鸟瞰/荷兰角（图标）
  - 运镜：静止/横摇/纵摇/推拉/变焦/跟拍（动画预览）

#### Compose 节点
- 视频预览播放器
- 轨道概览：视频轨 + 音频轨 + 字幕轨
- [合并视频] [导出剪映工程] [下载 MP4] 按钮
- 导出设置：分辨率/帧率/编码

### 时间线视图

```
┌──────────────────────────────────────────┐
│ 时间轴标尺  0:00  0:05  0:10  0:15 ...  │
├──────────────────────────────────────────┤
│ 视频轨 │[镜头1][镜头2][镜头3][镜头4]...  │
│ 音频轨 │[配音1  ][配音2  ][配音3  ]...   │
│ BGM轨  │[BGM ─────────────────────]      │
│ 字幕轨 │[字幕1][字幕2][字幕3]...         │
├──────────────────────────────────────────┤
│ 选中镜头属性：                            │
│ 时长: [2.5s] 转场: [淡入淡出▾]           │
│ [预览] [替换] [重新生成]                  │
└──────────────────────────────────────────┘
```

- 拖拽调整镜头顺序和时长
- 双击镜头 → 弹出编辑面板
- 转场效果拖放到镜头间

## 5. 小红书图文转视频 `/redbook`

### 布局：左右分栏

```
┌──────────────────┬───────────────────────┐
│ 输入区            │ 预览区                 │
│                  │                       │
│ ── 图片上传 ──    │  ┌─────────────────┐  │
│ [拖拽上传图片]    │  │                 │  │
│ [图1][图2][图3]   │  │   视频预览       │  │
│                  │  │   (9:16)        │  │
│ ── 事件描述 ──    │  │                 │  │
│ [今天去了xxx...]  │  │                 │  │
│                  │  └─────────────────┘  │
│ ── AI 文案 ──     │                       │
│ [生成爆款文案]    │  生成的文案：           │
│                  │  1. 第一句...          │
│ ── 配音选择 ──    │  2. 第二句...          │
│ [甜美女声▾][▶]   │  3. 第三句...          │
│                  │                       │
│ ── BGM ──        │  [下载MP4] [分享]      │
│ [轻快▾] [▶试听]  │                       │
│                  │                       │
│ [一键生成视频 →]  │                       │
└──────────────────┴───────────────────────┘
```

### 交互流程
1. 上传图片（支持拖拽，最多 20 张）→ 自动排序
2. 输入事件描述 → 点击「生成爆款文案」→ AI 生成 6-10 句
3. 文案可手动编辑、调整顺序
4. 选择配音和 BGM
5. 点击「一键生成视频」→ 右侧实时预览进度
6. 完成后可下载/分享

## 7. 素材库 `/assets`

### 布局：Tab 切换

```
[视觉风格] [配音库] [Prompt模板] [我的素材]
```

- 视觉风格：网格卡片，预览图 + 名称 + 分类筛选
- 配音库：列表，名称 + 性别 + 语言 + 试听按钮
- Prompt 模板：列表，模板名 + 分类 + 编辑/复制
- 我的素材：用户上传的图片/视频/音频，支持文件夹管理

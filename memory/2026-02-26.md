- **UI/样式**: Tailwind CSS + Radix UI + Lucide Icons
- **国际化**: next-intl（en / zh-cn）

- **部署**: 双区域（cn.drama.land 中国线路，drama.land 国际线路）
- **公司主体**: PawLogic Limited

### 画布页面技术栈
- **核心画布库**: React Flow (@xyflow/react) — 节点流程图引擎
- **状态管理**: Zustand
- **画布功能**: MiniMap, Controls, Panel, Handle, fitView, zoomIn/zoomOut, setViewport
- **动画**: Lottie
- **底层**: d3（React Flow 依赖的 d3-zoom/d3-selection 等）
- **路由**: `[locale]/(dashboard)/canvas/page`

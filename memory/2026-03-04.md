## 技术经验沉淀

### Docker 多阶段构建最佳实践
```dockerfile
# Builder 阶段：编译代码
FROM maven:3.9.9-eclipse-temurin-21-alpine AS builder
WORKDIR /build
COPY pom.xml .
COPY */pom.xml ./
RUN mvn dependency:go-offline -B
COPY */src ./*/src
RUN mvn clean package -DskipTests -B

# Runtime 阶段：只复制 JAR
FROM eclipse-temurin:21-jre-alpine
WORKDIR /app
COPY --from=builder /build/duo-video-api/target/*.jar app.jar
ENTRYPOINT ["java", "-jar", "app.jar"]
```

### Alpine Linux Python 环境问题
- 问题：PEP 668 限制，不允许直接 pip 安装到系统 Python
- 解决方案：
  1. 使用 `--break-system-packages`（快速但不推荐）
  2. 使用虚拟环境（推荐但复杂）
  3. 使用 `apk add py3-包名`（如果包可用）

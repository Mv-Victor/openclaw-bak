### Docker 多阶段构建
- Builder 阶段：Maven 编译，下载依赖
- Runtime 阶段：只复制 JAR，不暴露源代码
- 系统依赖：apk add + pip install（使用 --break-system-packages）

- 选中态使用扩散阴影更贴近 Drama.Land 风格

## storageService 初始化问题修复（2026-03-05 20:40 GMT+8）

### 问题背景
- 栋少指出：客户拿到的只有 Docker 镜像，没有源代码
- 客户需要通过 API 直接调用，无法使用 Maven 测试
- 之前的测试方式不符合实际使用场景

### 问题原因
- `JianyingBuilder.storageService` 是 static 字段
- Spring 无法自动注入 static 字段
- 导致 API 调用时 storageService 为 null

### 修复方案
```java
@Autowired
private StorageService injectedStorageService;

@PostConstruct
public void init() {
    storageService = injectedStorageService;
}
```

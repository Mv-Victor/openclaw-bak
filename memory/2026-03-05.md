## 自动截图圈选功能实现（11:15-11:20）

### 需求
栋少要求实现自助截图并用红色方框圈选图片关键部分的功能，用于处理小红书图文笔记。

### 实现方案
1. 使用 agent-browser CLI 进行网页截图
2. 用 Claude 视觉能力分析图片，识别关键区域坐标
3. 用 Python + Pillow 在图片上绘制红色方框

### 技术细节
- 工具脚本：`/root/.openclaw/workspace-g/scripts/highlight-key-areas.py`
- 依赖：Pillow (已安装到 python3.12)
- Python 环境问题：python3 指向 3.11，pip3 指向 3.12，需用 python3.12 执行脚本
- 坐标格式：JSON 数组，每个元素包含 x, y, width, height, label

### 工作流程
```bash
# 1. 截图
agent-browser open <url>
agent-browser screenshot /tmp/original.png

# 2. 分析并生成坐标 JSON
# (由 Claude 视觉分析生成 boxes.json)

# 3. 绘制方框
python3.12 highlight-key-areas.py input.png output.png boxes.json
```

### 限制
- 小红书需要登录才能查看笔记详情，测试时只截到了登录页
- 如需处理真实笔记内容，需要先登录或提供已登录的截图

### 验证结果
成功在小红书登录页截图上圈选了三个关键区域：品牌区、登录表单区、其他登录方式区。

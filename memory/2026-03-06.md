**问题 1**：字幕样式（黄色填充黑色描边）不生效

**原因**：API 参数格式错误 - 扁平化参数未被正确接收

**修复方案**：使用 `style` 对象格式
```json
// ❌ 错误格式（扁平化）
{
  "fontSize": 8,
  "fontColor": "#FFDE00",
  "strokeColor": "#000000",
  "strokeWidth": 60
}

// ✅ 正确格式（style 对象）
{
  "style": {
    "fontSize": 8,
    "fillColor": "#FFDE00",
    "strokeColor": "#000000",
    "strokeWidth": 60
  }
}
```

**问题 2**：字幕和水印位置不对（应该在底部/顶部居中）

## 技术经验沉淀

### Docker 镜像打包素材库
- **问题**: 挂载卷会覆盖镜像内目录
- **解决**: 素材库打包到镜像内，容器启动时不挂载该目录
- **最佳实践**: 
  ```dockerfile
  COPY doc/ /app/doc/  # 打包素材到镜像
  ```
  ```bash
  # 启动时不要挂载 /app/doc
  docker run ...  # 不加 -v $(pwd)/doc:/app/doc
  ```

### Spring Boot static 字段注入
- **问题**: Spring 无法直接注入 static 字段
- **解决**: 使用 `@Autowired` + `@PostConstruct` 模式
  ```java
  @Autowired
  private StorageService injectedStorageService;
  
  @PostConstruct
  public void init() {
      storageService = injectedStorageService;
  }
  ```

### 分镜设计原则
1. 图片时长 = 该图所有字幕时长之和（字读完才切图）
2. 字幕时长 = 字数 × 0.2s，最低 1500ms
3. 字幕位置：底部中央（positionY=750）
4. 水印位置：顶部中央（positionY=-800）
5. BGM 时长：与图片总时长一致
6. 表情包插入：情绪高潮点，不要都放末尾

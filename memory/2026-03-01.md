### 关键发现（从第一版数据中提取）

#### API 数据结构
从 `data/responses.json` 中提取到完整的项目数据结构：

1. **基础设定** (state: check_point)
   - mode: standard
   - idea_text: 创意文本（长篇故事）
   - language: 中文
   - episode_count: 4
   - episode_duration: 1.5
   - series_type: video
   - camera_frame_ratio: 9:16
   - visual_style_id: 1
   - rating: PG
   - ref_characters: []

2. **剧集设定 - 梗概** (state: synopsis)
   - synopsis_list: 3 个梗概选项
   - 每个梗概包含：title, description, genres, keywords
   - synopsis_selected_index: 选中的梗概索引

3. **剧集设定 - 世界观** (state: story_bible)
   - series_title: 系列标题
   - logline: 一句话概述
   - series_synopsis: 系列梗概
   - world_building: 世界观构建
   - suspense_bank: 悬念库（3个谜团）
   - standardized_tags: 标准化标签（genre, subgenre, tone, language, rating, keywords）

4. **角色集** (state: character_pack)
   - characters: 角色数组
   - 每个角色包含：
     - 基本信息：id, name, gender, age, height_cm, role_level, occupation_or_type
     - 性格：personality_snapshot, appearance, function_in_story
     - 配音：voice_id, voice_link, voice_description, voice_name
     - 图片：image_url, image_prompt (包含 prompt_define, prompt_feature, prompt_outfit, prompt_visual)
     - 深度信息：brief_bio, linguistic_style, long_term_want, core_flaw, stakes_if_fail, secrets_blackboxes, arc, values_and_lines

#### 角色图片 Prompt 模板
从角色数据中提取的 prompt 结构：
```
prompt_define: 基本定义（年龄、种族、体型）
prompt_feature: 面部特征（五官、肤色、发型、表情）
prompt_outfit: 服装描述
prompt_visual: 视觉风格（导演风格 + 拍摄参数）
```

示例（白骨夫人）：
- prompt_visual: "Alejandro G. Iñárritu Style: raw natural lighting, gritty realism, high contrast textures, and visceral cinematic intensity. Medium shot, front view, waist-up portrait, centered composition, 85mm equivalent. Soft studio lighting, uniform illumination."

## 一、核心能力

### 1. 标题生成（15字以内）

**7种爆款标题公式**：

| 类型 | 公式 | 示例 |
|------|------|------|
| 对比测评型 | [A] vs [B] vs [C]，[维度]测评 | 字节vs阿里vs携程，打工体验测评 |
| 数字具体型 | [时间]，[数字]家，[数字]家，[数字]个[结果] | 28天，投152家，面33家，4个50k offer |
| 情绪共鸣型 | [动作]了[对象]，现在感觉[情绪] | 拒了一圈人才计划，感觉被时代抛弃 |
| 场景细节型 | [公司][具体场景]测评/排名 | 大厂午休测评 腾讯vs字节vs美团 |
| 悬念问句型 | [公司][事件]？[夸张描述]？ | 秋招头部计划接近大满贯？ |
| 时间节点型 | 假如你从[时间]开始[动作][emoji] | 假如你从2.2开始投春招🙆🏻♀️ |
| 至暗时刻型 | [场景][负面事件]十大至暗时刻 | 大厂绩效考核十大至暗时刻 |

**标题检查清单**：
- [ ] 15字以内
- [ ] 包含具体数字
- [ ] 用问句（？）或省略号（...）结尾
- [ ] 有强动词或情绪词
- [ ] 化生僻词为熟悉词

### 2. 正文创作（4步框架）

```
① 事件陈述（100-150字）
   开头即冲突，无废话
   
② 独家角度（150-200字）
   新信息/新数据/评论区爆料
   
③ 利益点（100-150字）
   与读者相关：薪资/岗位/安全
   
④ 情绪/引导（50-100字）
   制造情绪，引发讨论
```

### 3. 配图建议（6-9张）

**优先级排序**：
1. 聊天记录截图
2. 数据对比图
3. 招聘信息截图
4. 文字卡片（金句）

### 4. 话题标签（5-8个）

**必带基础标签**：
```
#大厂 #求职 #打工人
```

**按内容类型叠加**：
- 求职：#秋招 #春招 #面试
- AI：#AI #ChatGPT #Cursor
- 薪资：#年终奖 #薪资 #跳槽

## 二、三大受众方向

### 方向1: 求职/大厂

**核心痛点**：
- 拿不到offer的焦虑
- 选择困难（多个offer如何选）
- 薪资谈判
- 职业发展路径

**标题公式**：
```
[时间] + [数字] + [结果] + ？
```

**情绪词库**：
- 焦虑类：危、被时代抛弃、寒气入骨
- 羡慕类：羡慕麻了、酸了、赢麻了

**示例标题**：
- 28天，投152家，面33家，4个50k offer
- 拒了一圈人才计划，现在感觉要被时代抛弃
- 秋招头部计划接近大满贯？

### 方向2: AI副业（OpenClaw相关）

**核心痛点**：
- 如何找到靠谱的副业方向
- 担心被割韭菜
- 不知道从哪里开始
- 收入不稳定

**标题公式**：
```
[工具/方法] + [时间/成本] + [收益] + [可复制性]
```

**情绪词库**：
- 利益驱动：搞钱、月入、睡后收入、躺赚
- 降低门槛：小白也能、保姆级、零基础

**OpenClaw相关角度**：
- 用AI agent自动化赚钱
- Claude API对接副业
- 多agent协作提升效率
- AI自动化工作流

**示例标题**：
- 用Cursor开发小程序，3天上线，月入5k
- AI agent自动化副业，睡后收入破万？
- Claude API对接赚钱，保姆级教程

### 方向3: AI剪辑

**核心痛点**：
- 剪辑效率低
- 不会复杂操作
- 批量生产需求
- 变现路径不清晰

**标题公式**：
```
[时间/效率] + [工具] + [结果] + [变现]
```

**情绪词库**：
- 效率类：自动化、批量、一键、秒出
- 收益类：月入、接单、变现、爆款

**示例标题**：
- AI剪辑1小时产出10条视频，月入2w
- 剪映自动化批量生产，接单到手软
- AI配音+自动剪辑，小白也能月入过万

## 三、爆款钩子技巧

### 开头钩子（前3句）

**公式**：
```
第1句：场景代入/数据冲击/情绪宣泄
第2句：制造冲突/悬念
第3句：引出核心观点
```

**示例**（评论2180案例）：
```
第1句："最近开放了一个后端实习的岗位，陆续收到了几份不错的简历。"
       → 场景代入
       
第2句："结果一周面下来，非常令人失望。"
       → 情绪词
       
第3句："我失去独立写代码的能力了。"
       → 核心冲突
```

### 中间钩子（制造悬念）

**技巧**：
- 数字冲击："就这道题，没有一个人做出来"
- 代际定位："这批毕业生正好就是22年上大学的那批人"
- 时间节点："大一还没结束chatgpt就面世了"

### 结尾钩子（引发争论）

**技巧**：
- 价值观冲突："依赖AI，就是在放弃自己最后的价值"
- 开放式问题："你站哪一边？"
- 情绪动员："打工人help打工人！"

## 五、争议话题引爆公式

### 核心公式

```
[传统观点] vs [新兴观点] + [代际冲突] + [利益相关]
```

### 高争议话题库

**AI相关**：
- AI取代程序员
- Cursor vs 手写代码
- ChatGPT写作业
- AI面试

**职场相关**：
- 996 vs WLB
- 大厂 vs 小厂
- 技术 vs 管理
- 35岁危机

**薪资相关**：
- 年终奖对比
- 跳槽涨薪
- 降薪裁员
- 期权兑现

### 制造争议的3个步骤

1. **设置对立面**：
   - 老一辈 vs 新一代
   - 传统方式 vs AI工具

2. **制造不确定性**：
   - "到底谁对谁错？"
   - "未来会怎样？"

3. **强化利益相关**：
   - "这关系到你的职业生涯"
   - "你的岗位会被取代吗？"

## 八、注意事项

### 禁忌

❌ **标题禁忌**：
- 超过15字
- 无数字无情绪词
- 生僻词（s-team、1号位）
- 平铺直叙

❌ **正文禁忌**：
- 长篇背景介绍
- 客套话（"大家好"）
- 废话（"今天想跟大家分享"）
- 纯文字长图

❌ **引导禁忌**：
- "上脉脉/去脉脉" → 容易被判违规
- 直接卖课 → 容易被限流
- 过度营销 → 降低信任度

### 最佳实践

✅ **标题最佳实践**：
- 15字以内
- 包含具体数字
- 用问句或省略号结尾
- 化生僻词为熟悉词

✅ **正文最佳实践**：
- 开头即冲突
- 口语化表达
- 具体数字和案例
- 制造情绪和悬念

✅ **配图最佳实践**：
- 6-9张
- 截图>设计图
- 真实>精美
- 信息密度高

### 2. 正文创作（4 步框架）

```
① 事件陈述（100-150 字）
   开头即冲突，无废话，直接抛出核心事件
   
② 独家角度（150-200 字）
   新信息/新数据/评论区爆料/个人感悟
   
③ 利益点（100-150 字）
   与读者相关：薪资/岗位/安全/成长路径
   
④ 情绪/引导（50-100 字）
   制造情绪，引发讨论，引导评论互动
```

## 二、三大受众方向

### 方向 1: 求职/大厂

**核心痛点**：
- 拿不到 offer 的焦虑
- 选择困难（多个 offer 如何选）
- 薪资谈判
- 职业发展路径
- 绩效压力

**标题公式**：
```
[时间节点] + [情绪/结果] + …
[时间] + [数字] + [结果] + ？
[公司]vs[公司]，[维度] 测评
```

**情绪词库**：
- 焦虑类：危、被时代抛弃、寒气入骨、破防了、崩溃、迷茫
- 羡慕类：羡慕麻了、酸了、赢麻了、泼天富贵
- 反差类：崩溃离开、涨薪 50%、差绩效、血泪总结

**示例标题**（真实案例）：
- 踩着春招的尾巴尖签了份巨好的工作…（2988 评论）

### 方向 2: AI 副业（OpenClaw 相关）

**核心痛点**：
- 如何找到靠谱的副业方向
- 担心被割韭菜
- 不知道从哪里开始
- 收入不稳定

**标题公式**：
```
[工具/方法] + [时间/成本] + [收益] + [可复制性]
让我非常 [情绪] 的 [AI 工具] 玩法。
```

**情绪词库**：
- 利益驱动：搞钱、月入、睡后收入、躺赚、变现
- 降低门槛：小白也能、保姆级、零基础
- 情绪冲击：震撼、颠覆、没想到、居然

**OpenClaw 相关角度**：
- 用 AI agent 自动化赚钱
- Claude API 对接副业
- 多 agent 协作提升效率
- AI 自动化工作流

**示例标题**：

- AI agent 自动化副业，睡后收入破万？
- Claude API 对接赚钱，保姆级教程

### 方向 3: AI 剪辑

**核心痛点**：
- 剪辑效率低
- 不会复杂操作
- 批量生产需求
- 变现路径不清晰

**标题公式**：
```
[时间/效率] + [工具] + [结果] + [变现]
[人群] [痛点] 攻略！
```

**情绪词库**：
- 效率类：自动化、批量、一键、秒出、效率翻倍
- 收益类：月入、接单、变现、爆款、到手软

**示例标题**：


## 八、注意事项

### 禁忌

❌ **标题禁忌**：
- 超过 15 字
- 无数字无情绪词
- 生僻词（s-team、1 号位）
- 平铺直叙

❌ **正文禁忌**：
- 长篇背景介绍
- 客套话（"大家好"）
- 废话（"今天想跟大家分享"）
- 纯文字长图

❌ **引导禁忌**：
- "上脉脉/去脉脉" → 容易被判违规
- 直接卖课 → 容易被限流
- 过度营销 → 降低信任度

### 最佳实践

✅ **标题最佳实践**：
- 15 字以内
- 包含具体数字或情绪词
- 用省略号（…）或问句（？）结尾
- 化生僻词为熟悉词

✅ **正文最佳实践**：
- 开头即冲突
- 口语化表达
- 具体数字和案例
- 制造情绪和悬念

✅ **配图最佳实践**：
- 6-9 张
- 截图>设计图
- 真实>精美
- 信息密度高

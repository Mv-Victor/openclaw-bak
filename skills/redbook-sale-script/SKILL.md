## 2. 标题写法

### 原则
- **15 字以内**，简短有力
- 结尾用**问句**（吸引点击）或**省略号**（制造悬念）
- 用**强动词**：喊话、要变天、炸了、开了、被逼、疯抢……
- 带**具体数字**：36 个月、10w、3 万、打骨折
- **化生僻词为熟悉词**：s-team → 最高管理层；1 号位 → 最高负责人

### 标题公式

```
[公司名] + [强动词/事件] + [数字/结果] + ？/ ...
```

## 3. 正文结构（4 步框架）

```
① 事件陈述
    ↓
② 脉脉独家新角度（脉脉上的新信息/评论区爆料/人才流动数据）
    ↓
③ 利益点（与我有关感：薪资/福利/岗位/职场安全）
    ↓
④ 进站引导
```

### 写作要点

**品牌词前置**
- 开头第一句植入"脉脉"：`刷到脉脉上xxx` / `刷脉脉xxx同事圈看到` / `某脉上早有爆料`

**口语化 & 与我有关感**
- 讲故事，接地气，代替打工人抒发情绪
- 强化薪资待遇、福利变化的具体对比
- 强调夸张数字：年终发 36 个月 / 近 4 年最高 / 大裁员 3 万

**情绪词库**

| 场景 | 情绪词 |
|------|--------|
| 追热点 | 最新消息、突发！刚刚、全网最全、速看、放大招、内幕流出、震撼 |
| 负面爆料/吐槽 | 天塌了、我是没想到、太难了、不是？、好手段啊、据传、炸裂、重磅、太抽象、离谱、破防了 |
| 福利/涨薪/期权 | 羡慕麻了、我酸了、打工人天花板、打工皇帝、这泼天富贵啥时候轮到我、赢麻了 |
| 通用强情绪 | 天塌了、离大谱、破防了、寒气入骨、羡慕麻了、酸了 |

**动词库**（突出前后变化，自带情绪）

退出 / 干掉 / 离开 / 被撵跑 / 飞升 / 空降 / 跑路 / 急眼 / 被逼XX / 狂招 / 抢疯 / 危 / 泡汤

## 4. 进站引导写法（按强弱分类）

### ❌ 禁忌写法
> 上脉脉 / 去脉脉 / 脉脉搜 → **容易被判违规引导第三方平台**

### ✅ 正确植入方式
前面正文植入：`刷脉脉看到了xxx` / `某脉上早有爆料`
结尾引导：`搞xx的老铁们懂得都懂，抓紧窗口期上车了`

---

### 弱引导：更多吃瓜

**适用**：热点事件、奇葩管理、吐槽贴

| 写法 | 示例 |
|------|------|
| 评论区热闹 | 脉脉评论区吵翻了！ |
| 新爆料预告 | 脉脉上又有新爆料，讨论非常精彩... |
| 同事圈猛料 | 脉脉的同事圈真的有点猛料啊！大家没事多刷刷未雨绸措下！ |

---

### 中引导：干货学习 / 社交链接

**适用**：牛人履历、职场技巧、晋升绩效节点

| 写法 | 示例 |
|------|------|
| 干货学习 | 不知道怎么给下属打绩效的新手 leader 都去脉脉学，搜绩效 |
| 履历参考 | 太长了没截完，在改简历的都建议脉脉找原帖学习下、抄抄作业 |
| 加好友 | 还能加好友，万一以后有用呢 |
| 大佬互动 | 大佬每天活跃在脉脉，我已经去求好友位了 |

> ⚠️ 干货引导要精准，有明确 benefit，不要泛泛说"脉脉上职场技巧挺多"

---

### 强引导：岗位机会 / 年终奖情报 / 曝光投诉

**适用**：裁员、人才流动、年终奖开奖、职场不公

| 类型 | 写法 | 示例 |
|------|------|------|
| 岗位机会 | highlight 高薪在招岗位 | 某司在脉脉有大量高薪岗位在招；HR 在脉脉高度活跃，狂招人 |
| 岗位机会 | 内推帖 | 某司在脉脉有大量内推帖（正职+实习） |
| 年终奖情报 | 对齐水位 | 你司的年终奖在脉脉上早就传开了，到底有没有被压？去脉脉搜一下公司名+年终 |
| 年终奖情报 | 晒开奖 | 好多xx员工都在某脉上晒开奖情况啦，去看看你是啥水位 |
| 曝光投诉 | 号召行动 | 脉脉投诉是真·有效啊，家人们遇到类似的（职场不公）事件吗，也可以在脉脉曝光他们！ |
| 声援支持 | 情绪动员 | 手段太毒了！家人们速来脉脉声援，打工人 help 打工人！ |

## 5. 话题 Tag 推荐

### 必带（每篇都加）
```
#互联网大厂  #大厂  #打工人
```

### 按内容类型叠加

| 内容类型 | 叠加 Tag |
|----------|----------|
| 年终奖 | #年终奖 #绩效 #职场 + #具体公司名 |
| 裁员/人才流动 | #职场 #求职 + #具体公司名 |
| 奇葩管理/吐槽 | #职场 #抽象 + #具体公司名 |
| 人物离职/履历 | #求职 #公关（按职能） + #具体公司名 |
| 热点追踪 | #职场 + #具体公司名 + #事件关键词 |

> 具体公司名单独一个 tag，如 #腾讯 #百度 #奇瑞 #米哈游

## 6. 完整创作 Prompt 模板

```
你是一名小红书爆文写手，专门为职场社区脉脉创作热点笔记。

【任务】
根据以下热点事件，创作一篇小红书笔记。

【热点事件】
{在此填入事件描述，包括：公司名、事件内容、关键数字、脉脉上的相关信息}

【创作要求】

标题：
- 15 字以内
- 用问句（？）或省略号（..）结尾
- 包含公司名 + 强动词 + 数字/结果
- 化生僻词为大众熟悉的表达

正文结构（严格按此顺序）：
1. 事件陈述：开头第一句植入"脉脉"品牌词，口语化讲述事件
2. 脉脉独家角度：引用脉脉同事圈/评论区的新信息、人才流动数据、爆料截图描述
3. 利益点：强调与读者的关联（薪资/岗位/职场安全），用具体数字和情绪词
4. 进站引导：根据内容类型选择强/中/弱引导，不能出现"上脉脉/去脉脉/在脉脉搜"

语气要求：
- 口语化，像朋友聊天
- 多用情绪词：羡慕麻了/酸了/天塌了/离大谱/破防了/炸裂
- 多用强动词：干掉/被撵跑/飞升/空降/狂招/抢疯
- 代替打工人抒发怨气/羡慕/焦虑

话题 Tag：
- 必带：#互联网大厂 #大厂 #打工人
- 加上具体公司名和内容类型相关 tag（年终奖/职场/求职等）
- 共 5-8 个 tag

【内容类型】（请选择最匹配的一种）
□ 年终奖爆料  □ 裁员/人才流动  □ 奇葩管理  □ 吃瓜爆料  □ 人物离职/履历
```

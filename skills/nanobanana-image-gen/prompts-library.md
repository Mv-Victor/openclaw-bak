## 使用技巧

### 1. 参数化 Prompts
使用 `{variable}` 占位符，方便重复使用和定制。

### 2. 组合风格
可以组合多种风格，例如："科幻 + 赛博朋克 + 电影级画质"。

### 3. 指定比例
- 1:1 - 正方形（Instagram、头像）
- 16:9 - 横版（YouTube、文章封面）
- 9:16 - 竖版（小红书、手机壁纸）
- 4:3 - 传统比例

### 4. 添加细节
越详细的描述，生成的图片越符合预期。包括：
- 颜色
- 光照
- 构图
- 风格
- 情绪

### 5. 迭代优化
如果第一次生成的图片不满意，可以：
- 调整 prompt 描述
- 添加更多细节
- 尝试不同的风格
- 重新生成

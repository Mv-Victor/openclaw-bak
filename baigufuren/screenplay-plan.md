# 《白骨夫人》短剧剧本分场方案

## 原作信息
- 原作：《白骨夫人》左左薇拉，第九届光年奖微小说一等奖
- 核心设定：人类是"脑人"和"骨人"的共生体
- 原文8章，约6000字

## 情绪弧线设计

```
觉醒(0) → 恐惧(2) → 温暖(1) → 震撼(3) → 愤怒(4-PEAK) → 悲伤(3) → 释然(0)
```

节奏型：classic-arc（经典五幕）

## 分场方案（7场）

### 第1场：棺中觉醒（Setup / 困惑）
- 对应原文：第1章
- 时长：6-7s
- 内容：黑暗中醒来，破棺而出，发现自己变成了骨架，身旁是自己的血肉之躯
- 情绪：困惑 → 恐惧，intensity 0→2
- 关键画面：
  - 棺材内部特写，手指摸索缝隙
  - 骨手破开"棺材"（实际是自己的皮肉）
  - 低头看见白森森的骨骼——身份崩塌的瞬间
  - 乱葬岗全景，月光下唯一站立的骷髅
- 镜头语法：ECU→中景拉出，暗调，手持微晃
- BGM：低频嗡鸣 + 心跳声渐强

### 第2场：明月引路（Rising / 温暖）
- 对应原文：第2-3章
- 时长：5-6s
- 内容：悟空差点打她，唐三藏制止；明月解释"人是共生之物"，脱下自己的皮展示骨人身份，邀她回五庄观
- 情绪：恐惧 → 温暖，intensity 2→1
- 关键画面：
  - 悟空铁棍砸地，骷髅被震翻
  - 唐三藏伸手，温和的目光
  - 明月褪去肌肤，露出骷髅——"白骨，和我回家吧"
- 镜头语法：动态跟拍→静态中景，色调从冷转暖
- BGM：紧张弦乐 → 温暖木管

### 第3场：人参果与骨人复国（Rising / 震撼）
- 对应原文：第4章
- 时长：5-6s
- 内容：五庄观，镇元子设宴，人参果覆体变回人形；镇元子揭示真相——人参果是人造脑人，骨人才是世界的主人
- 情绪：好奇 → 震撼，intensity 1→3
- 关键画面：
  - 人参果如卷轴展开，覆上骨架，经脉攀附
  - 铜镜中映出九分顾瑶的容颜——短暂的美丽
  - 镇元子冷冽的面容："这世界终有一日将归属我族"
- 镜头语法：特写→全景推拉，暖调突转冷调
- BGM：神秘东方弦乐 → 低沉号角

### 第4场：上元夜·背叛（Climax / 愤怒 - PEAK）
- 对应原文：第5章
- 时长：4s（climax压缩）
- 内容：白骨偷回武威城，上元灯会，谢府中堂——发现谢长安与突厥人觥筹交错，叛变卖国，称她为"区区贱婢"；白骨怒推门，谢长安挽弓射箭，正中脊柱"谢氏夫人"四字
- 情绪：期待 → 愤怒爆发，intensity 3→4（PEAK）
- 关键画面：
  - 上元花灯满城，赤红灯火
  - 中堂内谢长安与突厥人大笑——"区区贱婢，不足挂齿"
  - 白骨推门而入，怒喝"谢长安！"
  - 箭破空而来，穿透人参果皮，射入脊柱——慢动作
  - 花灯意象：灯芯被射灭
- 镜头语法：ECU怼脸→快速正反打→慢动作特写（箭入骨）
- BGM：打击乐爆发 + 弦乐撕裂 → 箭中骨的瞬间静默

### 第5场：骨人的苦难（Falling + Silence Window / 悲伤）
- 对应原文：第7章（岩穴段落）
- 时长：8-9s（silence window，情绪留白）
- 内容：明月带白骨进入五庄观地下岩穴，看到骨人千年来的苦难——被烧死的小骨人、痛哭的骨人、被打碎的骨人情侣；明月讲述骨人千年摸索史
- 情绪：愤怒余波 → 深沉悲伤，intensity 4→3（PEAK→FALLING）
- 关键画面：
  - 岩穴深处，钟乳石与人参果树根纠缠
  - 石案上的解剖场景（暗示性，不直接展示）
  - 被烧成炭的小骨人——"偷溜回家，被阿娘当鬼怪烧死了"
  - 双手紧握、碎成碎片的骨人情侣
  - 血花喷溅，所有骨人静止——时间凝固的悲伤
- 镜头语法：缓慢推轨 + 长镜头，pull-out留白
- BGM：前2秒完全静默 → 稀疏古琴单音 → 低沉大提琴

### 第6场：花灯之悟（Resolution / 释然）
- 对应原文：第8章
- 时长：6-7s
- 内容：白骨回忆阿娘的话——"灯笼灯笼，先有灯才有笼，笼中烛火一熄，灯笼便成废纸弃木"；她做出选择：不加入骨人复国，选择共生
- 情绪：悲伤 → 释然，intensity 3→0
- 关键画面：
  - 阿娘教她编花灯的温暖回忆（暖色调闪回）
  - 花灯意象：灯骨+灯面+烛火 = 完整的灯笼
  - 白骨在美人榻上，没了人参果，越来越难动弹
  - 悟空念出脊柱上的字："白……骨……夫……人……"
  - 白骨的"表情"——骨人没有肌肉，但观众能感受到她的微笑
- 镜头语法：柔焦闪回→静态中景→缓慢zoom in到脊柱刺字
- BGM：温暖古筝 + 笛声 → 渐弱至静

### 第7场：尾声·花灯（Coda）
- 对应原文：意象延伸
- 时长：3-4s
- 内容：上元灯会的花灯缓缓升空，一盏骨架花灯混在其中，烛火明亮
- 情绪：平静，intensity 0
- 关键画面：
  - 万千花灯升空的壮观全景
  - 其中一盏花灯的骨架清晰可见——但烛火最亮
  - 黑屏，出字幕
- 镜头语法：大全景→缓慢tilt up→fade to black
- BGM：环境音（风声+远处人声）→ 静

---

## 总时长预估
37-43s（7场），符合短剧节奏

## 视觉风格建议
- 整体色调：暗色系为主，花灯场景暖色点缀
- 美术风格：中国古风+科幻质感（骨骼的荧光绿火、人参果的生物发光）
- 核心视觉符号：花灯（贯穿全片的意象）

## 情绪编排要点
- 第4→5场是 PEAK→SILENCE 断崖（与《最后一通电话》实验验证的模式一致）
- 花灯意象串联全片：第1场（乱葬岗月光）→ 第4场（上元灯会）→ 第6场（阿娘的花灯）→ 第7场（升空的骨架花灯）
- 谢长安的两次射箭形成首尾呼应：初见射花灯（美好）→ 背叛射脊柱（毁灭）

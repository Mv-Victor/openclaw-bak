## 一、超级爆款案例深度拆解

### 案例1: "AI vs 传统编程能力"争议帖
- **数据**: 评论2180 | 点赞4720 | 收藏2328
- **作者**: 南科大曾学长（面试官视角）
- **发布时间**: 2025-01-02

#### 标题策略
- **无标题策略**：让正文第一句话成为钩子
- 开头即冲突："最近开放了一个后端实习的岗位，陆续收到了几份不错的简历。结果一周面下来，非常令人失望。"

#### 正文结构（完美符合4步框架）

```
① 事件陈述
   "我发现他们大部分都失去独立写代码的能力了"
   
② 独家新角度
   "22年上大学那批人，大一还没结束chatgpt就面世了，大二就有cursor了"
   
③ 利益点（与我有关感）
   "如果一个学生从上学的时候就完全依赖AI，自己放弃了独立写代码的能力，
    那就不是在用工具了，而且变成彻底的传声筒了"
   
④ 情绪引爆
   "未来有一天AI足够强大了，程序员这个职业也就彻底消失了"
```

#### 钩子技巧

**开头钩子**（前3句）:
1. "最近开放了一个后端实习的岗位" — 场景代入
2. "结果一周面下来，非常令人失望" — 情绪词
3. "我发现他们大部分都失去独立写代码的能力了" — 核心冲突

**中间钩子**（制造悬念）:
- "就这道题，没有一个人做出来" — 数字冲击
- "我猛然反应过来，这批毕业生正好就是22年上大学的那批人" — 代际定位

**结尾钩子**（引发争论）:
- "那么依赖AI，疏于学习和对自己的锻炼，就是在放弃自己最后的价值" — 价值观冲突

#### 情绪词库应用

| 类型 | 原文使用 | 脉脉方法论对应 |
|------|----------|----------------|
| 负面情绪 | "非常令人失望"、"完全不行" | 天塌了、我是没想到 |
| 强调程度 | "完全依赖"、"彻底的传声筒" | 离大谱、破防了 |
| 焦虑制造 | "程序员这个职业也就彻底消失了" | 寒气入骨、危 |

#### 评论区引爆点

**两派对立**（完美制造争议）:

**老一辈阵营**（支持作者）:
- 评论665赞："你可以不懂手搓代码，但是你不可以不懂数据结构与算法"
- 评论122赞："任何ai写的东西都是需要人工来核验的"

**新一代阵营**（反对作者）:
- 评论750赞："如果一个公司因为我不会手搓代码把我淘汰，说明这个公司没有办法很好的跟上时代的进度"
- 评论1199赞："已经划分成原教旨遗老和新世界AI两派了"

**中立/建设性**:
- 评论734赞："或许该转向考核，让他们检测ai写出来的代码有没有问题"
- 评论223赞："也许我们的考察方式落伍了，不应该考察写代码，而是考察是否能识别好代码坏代码"

| 步骤 | 应用率 | 典型表现 |

## 八、受众分层与内容策略

### 三大受众方向的差异化策略

#### 1. 求职/大厂方向

**核心痛点**:
- 拿不到offer的焦虑
- 选择困难（多个offer如何选）
- 薪资谈判
- 职业发展路径

**标题公式**:
```
[时间] + [数字] + [结果] + ？
例：28天，投152家，面33家，4个50k offer
```

**情绪词**:
- 焦虑类：危、被时代抛弃、寒气入骨
- 羡慕类：羡慕麻了、酸了、赢麻了

**引导方式**:
- 弱引导：评论区互通有无
- 中引导：简历学习、面试技巧
- 强引导：内推机会、岗位信息

#### 2. AI副业方向

**核心痛点**:
- 如何找到靠谱的副业方向
- 担心被割韭菜
- 不知道从哪里开始
- 收入不稳定

**标题公式**:
```
[工具/方法] + [时间/成本] + [收益] + [可复制性]
例：用Cursor开发小程序，3天上线，月入5k（附教程）
```

**情绪词**:
- 利益驱动：搞钱、月入、睡后收入、躺赚
- 降低门槛：小白也能、保姆级、零基础

**引导方式**:
- 强引导：工具推荐、教程链接、社群加入
- 中引导：案例学习、经验分享
- 避免：直接卖课（容易被判违规）

**OpenClaw相关角度**（栋少要求）:
- "用AI agent自动化赚钱"
- "Claude API对接副业"
- "多agent协作提升效率"
- "AI自动化工作流"

#### 3. AI剪辑方向

**核心痛点**:
- 剪辑效率低
- 不会复杂操作
- 批量生产需求
- 变现路径不清晰

**标题公式**:
```
[时间/效率] + [工具] + [结果] + [变现]
例：AI剪辑1小时产出10条视频，月入2w（工具推荐）
```

**情绪词**:
- 效率类：自动化、批量、一键、秒出
- 收益类：月入、接单、变现、爆款

**引导方式**:
- 强引导：工具推荐、模板分享
- 中引导：技巧学习、案例拆解
- 弱引导：评论区交流

## 九、爆款笔记创作SOP

### Step 1: 选题（优先级排序）

1. **争议话题**（评论>1000）:
   - AI vs 传统能力
   - 大厂 vs 小厂
   - 996 vs WLB

2. **对比测评**（评论500-1000）:
   - 多公司对比
   - 多工具对比
   - 前后对比

3. **数字冲击**（评论300-500）:
   - 具体时间+数量+结果
   - 薪资数据
   - 涨幅对比

4. **场景细节**（评论200-300）:
   - 具体场景描述
   - 真实案例
   - 个人经历

### Step 2: 标题（15字以内）

**公式**:
```
[主体] + [强动词/数字] + [结果/悬念] + ？/...
```

**检查清单**:
- [ ] 15字以内
- [ ] 包含具体数字
- [ ] 用问句或省略号结尾
- [ ] 有强动词或情绪词
- [ ] 化生僻词为熟悉词

### Step 3: 开头（前3句）

**钩子公式**:
```
第1句：场景代入/数据冲击/情绪宣泄
第2句：制造冲突/悬念
第3句：引出核心观点
```

**禁忌**:
- ❌ 长篇背景介绍
- ❌ 客套话（"大家好"）
- ❌ 废话（"今天想跟大家分享"）

### Step 4: 正文（4步框架）

```
① 事件陈述（100-150字）
   - 具体场景
   - 关键数据
   - 核心冲突

② 独家角度（150-200字）
   - 新信息/新数据
   - 评论区爆料
   - 内部视角

③ 利益点（100-150字）
   - 与读者相关
   - 薪资/岗位/安全
   - 具体影响

④ 情绪/引导（50-100字）
   - 制造情绪
   - 引发讨论
   - 隐性引导
```

### Step 5: 配图（6-9张）

**优先级**:
1. 聊天记录截图
2. 数据对比图
3. 招聘信息截图
4. 文字卡片（金句）

**禁忌**:
- ❌ 纯文字长图
- ❌ 过度设计
- ❌ 无关配图

### Step 6: 标签（5-8个）

**必带**:
```
#大厂 #求职 #打工人
```

**按内容叠加**:
- 求职：#秋招 #春招 #面试
- AI：#AI #ChatGPT #Cursor
- 薪资：#年终奖 #薪资 #跳槽
- 公司：#具体公司名

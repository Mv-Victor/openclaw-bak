## dreamX 项目笔记
- **项目名**：duo-video — Java 视频编辑 SDK，通过生成剪映工程文件实现视频创作
- **技术栈**：Java 21 + Maven + Spring Boot 3.5.8
- **模块结构**：
  - `duo-server-base` — 基础工具库（文件、JSON、工具类）
  - `duo-video-base` — 核心数据模型（VideoProject、Material、Segment）+ Builder 链式 API
  - `duo-video-jy` — 剪映集成（将 VideoProject 转换为剪映工程格式）
  - `duo-video-api` — RESTful API 接口
  - `auto-jy` — 自动剪映工具（Mac/Win，领取任务、导出视频）
  - `skills` — AI Skills 集成

### 核心流程
1. 构建 `VideoProject`（JSON 或 Builder API）
2. `JianyingBuilder.build(project)` → 生成 `JianYingProjectInfo`（剪映草稿）
3. auto-jy 领取任务 → 剪映导出成品视频

### 重点文件
- `JianyingBuilderTest.java` — 三个测试方法：
  - `buildWithProjectJson()` — 从 JSON 加载 VideoProject 并构建
  - `buildWithProjectBuilder()` — 链式调用构建
  - `buildWithBuilder()` — 函数式构建（最完整，覆盖所有素材类型）
- `ProjectBuilder.java` — 链式/函数式 Builder 入口
- `JianyingBuilder.java` — 核心构建器，先构建素材再构建分镜
- `VideoProject.java` — 顶层数据模型

### 支持的素材类型
- 基础：视频、图片、文本、字幕、音频、文本模板
- 特效：特效音、贴纸、转场、画面特效、脸部特效
- 高级：绿幕抠图、复合片段、蒙板、LUT 滤镜、花字、文本预设样式

### 坐标系
- 原点 (0,0) = 视频中心
- X: 左负右正
- Y: 上正下负（与常规相反）
- 时间单位：毫秒
- 缩放单位：万分比（5000 = 50%）

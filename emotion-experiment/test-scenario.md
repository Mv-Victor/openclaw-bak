# 三版对比实验 - 测试剧本

## 测试场景：《最后一通电话》

### 剧情概要
深夜，一个年轻女人独自坐在昏暗的客厅里，手机屏幕亮起——是前男友打来的电话。她犹豫、接听、争吵、崩溃、最终挂断电话，走到窗前看着城市夜景，擦干眼泪。

### 分场剧本

**第1场 - 沉默的等待（情绪：平静→微妙不安）**
深夜的客厅，只有电视的微光。林晓坐在沙发上，手里握着一杯已经凉了的咖啡，目光空洞地看着前方。

**第2场 - 电话响起（情绪：不安→紧张）**
手机突然震动，屏幕亮起"陈默"两个字。林晓的手微微颤抖，她盯着屏幕，咬了咬嘴唇。

**第3场 - 犹豫接听（情绪：紧张→挣扎）**
电话持续响着。林晓深吸一口气，闭上眼睛，最终按下了接听键。"喂？"声音很轻。

**第4场 - 争吵爆发（情绪：压抑→愤怒爆发）**
林晓猛地站起来："你现在才想起来打电话？三个月了！"她的声音越来越大，眼眶泛红，手紧紧攥着手机。

**第5场 - 情绪崩溃（情绪：愤怒→悲伤崩溃）**
林晓的声音突然哽咽，她慢慢蹲下来，靠着沙发，泪水无声地流下。"你知道我等了多久吗……"

**第6场 - 决断挂断（情绪：悲伤→坚定）**
林晓擦了擦眼泪，深呼吸，声音恢复平静："别再打了。再见。"果断挂断电话。

**第7场 - 窗前释然（情绪：坚定→平静释然）**
林晓走到窗前，推开窗户，城市的夜风吹过她的脸。她望着远处的灯火，嘴角微微上扬。

---

## 情绪弧线

```
场次:  1    2    3    4    5    6    7
情绪: 平静→不安→紧张→愤怒→悲伤→坚定→释然
强度:  0    1    2    3    3    2    0
张力: ─────╱───╱──╱peak╲──╲─────
```

## 验收标准

### 版本A（基准 - master分支现有逻辑）
- 用现有 storyboard prompt 生成7场分镜
- 记录每场的 shot_type, movement, angle, image_prompt, video_prompt
- 作为对照组

### 版本B（镜头语法库集成）
- 集成 pkg/shotgrammar 到分镜生成流程
- 根据场景类型自动匹配 ShotRule
- 用 MotionHint 增强 video_prompt
- 对比：镜头选择是否更专业、motion prompt 是否更精准

### 版本C（+ 情绪编排引擎）
- 在B基础上，先分析全剧情绪弧线
- 用 EmotionToMotionStyle 驱动镜头语法选择
- 情绪强度影响 pacing、transition
- 对比：情绪节奏是否连贯、镜头语言是否匹配情绪变化

## 服务流程

### Step 1: 需求沟通（30 分钟）
- 了解你的使用场景（编程助手？内容创作？自动化任务？）
- 确定 Agent 角色定位（需要几个 Bot？各自职责？）
- 确认部署环境（服务器配置、网络环境）

### Step 2: 环境准备（1 小时）
- 帮你注册云服务器（或配置你自己的）
- 安装 Node.js、Git、浏览器依赖
- 配置防火墙和安全组

### Step 3: OpenClaw 部署（1 小时）
- 安装 OpenClaw CLI
- 配置 Gateway（openclaw.json）
- 配置飞书渠道（webhook + app ID）
- 配置双 Agent workspace

### Step 4: 技能安装（30 分钟）
- 安装基础技能（weather、summarize、agent-browser 等）
- 安装专业技能（python-pro、react-specialist、backend-architect 等）
- 配置自定义技能（根据你的需求）

### Step 5: 协作配置（30 分钟）
- 配置群聊监听（requireMention、mentionPatterns）
- 配置记忆系统（MEMORY.md、daily memory）
- 配置 Git 自动备份
- 配置定时任务（cron）

### Step 6: 交付验收（30 分钟）
- 现场测试群聊协作流程
- 测试任务派工和执行
- 测试记忆持久化
- 交付所有配置文档和踩坑经验

**总耗时**：3-4 小时
**交付物**：
- 可运行的 OpenClaw 双 Bot 系统
- 完整的配置文件（openclaw.json、SOUL.md、AGENTS.md 等）
- 踩坑经验文档（PDF + Markdown）

# DreamX Studio — 页面交互设计文档

> 版本: v1.0 | 日期: 2026-02-26 | 作者: 啾啾

---

## 1. 全局设计规范

### 1.1 设计语言

| 维度 | 规范 |
|------|------|
| 主色调 | 深色主题 `#0F0F14`，强调色 `#7C5CFC`（紫）+ `#FF6B35`（橙） |
| 字体 | 中文：思源黑体；英文：Inter；代码：JetBrains Mono |
| 圆角 | 卡片 12px，按钮 8px，输入框 8px |
| 间距 | 基础单位 4px，常用 8/12/16/24/32 |
| 动效 | 过渡 200ms ease-out，加载态用 Lottie 骨架屏 |

### 1.2 响应式断点

| 断点 | 宽度 | 布局 |
|------|------|------|
| Desktop | ≥1280px | 三栏（聊天+画布+属性面板） |
| Tablet | 768-1279px | 双栏（画布+可折叠侧栏） |
| Mobile | <768px | 单栏（Tab 切换） |

MVP 阶段优先 Desktop，Tablet 做基础适配，Mobile 暂不支持。

## 2. 页面结构总览

```
/                          → 首页（Landing Page）
/login                     → 登录/注册
/projects                  → 项目列表（我的工作台）
/canvas/:projectId         → 画布创作页（核心页面）
/canvas/:projectId/:episodeId → 单集详情页
/assets                    → 素材库（风格/配音/表情包/BGM）
/settings                  → 账户设置 + 订阅管理
```

这是整个平台最核心的页面，融合 Drama.Land 画布式 + Moyin 时间线双视图。

#### 布局（三栏）

```
┌────────┬──────────────────────────┬─────────┐
│        │ TopBar: 项目名 | 视图切换 │         │

│        │                          │         │
└────────┴──────────────────────────┴─────────┘
```

#### 3.3.1 画布视图（创作阶段）

React Flow 节点流程图，节点类型对应创作阶段：

```
[入口] → [基础配置] → [故事圣经] → [角色集] → [规划中心] → [剧本] → [场景设计] → [分镜设计] → [合成导出]
         CheckPoint   StoryBible  CharacterPack PlanningCenter Script  SceneDesign SegmentDesign Compose
```

节点状态视觉：
| 状态 | 样式 |
|------|------|
| 未开始 | 灰色边框，虚线 |
| 进行中 | 紫色边框，脉冲动效 |
| AI 生成中 | 紫色边框 + Lottie 加载动画 |
| 已完成 | 绿色边框，实线，✓ 图标 |
| 失败 | 红色边框，⚠ 图标，点击重试 |

节点间连线：已完成的连线为实线绿色，未完成为虚线灰色。

点击节点 → 右侧属性面板展开该节点的详情编辑区。

## 4. 全局交互组件

### 4.1 顶部导航栏（登录后）

```
┌─────────────────────────────────────────────────┐
│ [Logo] DreamX Studio    积分:4825 [STARTER▲] 🔔 👤│
└─────────────────────────────────────────────────┘
```

- Logo 点击 → `/projects`
- 积分显示实时余额，点击 → 积分明细
- 订阅标签点击 → 升级弹窗
- 🔔 通知: 生成完成、积分不足等
- 👤 头像下拉: 设置、帮助、退出

### 4.2 AI 生成进度条

所有 AI 生成操作统一使用 SSE 推送进度：

```
┌──────────────────────────────┐
│ 🎨 正在生成角色形象...        │
│ ████████████░░░░░░ 67%       │
│ 预计剩余 12s                  │
│                    [取消]     │
└──────────────────────────────┘
```

- 右下角 Toast 通知，不阻塞操作
- 多任务并行时，堆叠显示
- 完成后自动消失，画布节点状态更新

### 4.3 积分消耗确认弹窗

高消耗操作（>50 积分）弹出确认：

```
┌──────────────────────────────┐
│ ⚡ 确认消耗积分                │
│                              │
│ 操作: 生成视频片段             │
│ 消耗: 300 积分                │
│ 余额: 4825 → 4525            │
│                              │
│ [取消]          [确认生成]    │
└──────────────────────────────┘
```

```

│   ├── nodes: CanvasNode[]
│   ├── edges: Edge[]
│   ├── activeNodeId: string | null
│   ├── viewMode: 'canvas' | 'timeline'
│   └── actions: { selectNode, updateNode, addEdge }
├── chat: ChatStore
│   ├── messages: ChatMessage[]
│   ├── isStreaming: boolean
│   └── actions: { send, receive }
├── timeline: TimelineStore
│   ├── tracks: Track[]
│   ├── playhead: number (ms)
│   ├── zoom: number
│   └── actions: { addSegment, moveSegment, resize }
└── generation: GenerationStore
    ├── tasks: GenerationTask[]
    └── actions: { start, cancel, retry }
```

## 7. 错误处理与边界状态

| 场景 | 处理方式 |
|------|---------|
| 网络断开 | 顶部黄色 Banner "网络已断开，正在重连..." |
| AI 生成失败 | 节点标红 + Toast 提示 + 重试按钮 |
| 积分不足 | 弹窗提示 + 引导充值/升级 |
| 项目自动保存 | 每 30s 自动保存，右上角显示"已保存 ✓" |
| 并发编辑冲突 | MVP 不支持多人协作，单用户锁 |
| 大文件上传 | 分片上传 + 进度条，支持断点续传 |
